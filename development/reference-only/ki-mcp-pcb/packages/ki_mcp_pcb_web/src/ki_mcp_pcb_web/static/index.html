About

ki-mcp-pcb turns plain-text PCB descriptions into manufacturable KiCad projects. This viewer is the optional fourth delivery shape (see SPEC.md §2) — the other three are the Python library, the kimp CLI, and the MCP server for Claude Code.

Every panel here is a thin call into the same core library that powers kimp build. There's no logic duplication.
