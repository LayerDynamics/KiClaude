**Goal:** Drop the `simple` zone-fill fixture's XOR from 0.0123 mm² toward ≤ 0.01 mm² by making the min-thickness opening's round corners edge-aligned (matching KiCad/Clipper2 `DoRound`), without regressing `concave` (0.0010) or `thermal` (0.0047).

- Current primitive: `crates/cad/src/zones/boolean.rs:229` `minkowski_disc_primitives(ring, delta)` emits, per edge, a **symmetric** rectangle (`±n·delta`, lines 245-250) and, per vertex `a`, a **full** `disc_polygon(a, delta, OFFSET_ARC_SEGMENTS)` (line 252). `disc_polygon` (line ~264) already uses the maxError-based segment count (`KICAD_MAX_ERROR_MM = 0.005`, line 262), inscribed, phase-0.
- Clipper2 `DoRound` round join (KiCad's engine): per vertex, first arc point at `vertex + n_in·delta`, interior points by rotating the offset vector by `2π/steps_per_360` per step, exact final point at `vertex + n_out·delta`; vertices ON the arc (inscribed). Sweep angle = signed turn `atan2(cross(n_in,n_out), dot(n_in,n_out))`. Normal of edge p1→p2 is `(dy,-dx)` (right of travel).

- Reality check: KiCad snaps to integer nanometres; exact 0.01 in f64 may be unreachable. Success = `simple` ≤ 0.01 on all three fixtures; if it floors at e.g. 0.011, KEEP the 0.015 gate and document — do not fake it.

**Step 3: Write the helper + density fn**
```rust
/// Segments per radian for round-join arcs at radius `delta`, matching
/// KiCad/Clipper2's `m_MaxError` arc density: a full circle uses
/// `n = ceil(2π / (2·acos(1 − err/delta)))` segments (≥4), so the
/// per-radian rate is `n / (2π)`.
fn steps_per_rad(delta_mm: f64) -> f64 {
    if delta_mm <= KICAD_MAX_ERROR_MM {
        return 4.0 / std::f64::consts::TAU;
    }
    let inc = 2.0 * (1.0 - KICAD_MAX_ERROR_MM / delta_mm).acos();
    let n = (std::f64::consts::TAU / inc).ceil().max(4.0);
    n / std::f64::consts::TAU
}

/// Edge-aligned round-join "pie slice" at vertex `apex`, sweeping the
/// SHORT way from the incoming-edge normal `n_in` to the outgoing-edge
/// normal `n_out`, both unit vectors. Mirrors Clipper2 `DoRound`:
/// first ring vertex after the apex is `apex + n_in·delta`, interior
/// vertices rotate the offset vector by the signed sweep in
/// `steps_per_rad`-sized increments, final vertex is exactly
/// `apex + n_out·delta`. Winding-agnostic: a left turn sweeps the
/// convex (outer) side and fills the gap between edge rectangles; a
/// right turn sweeps the concave side and lands inside the union
/// (harmlessly absorbed). Returned as a closed polygon (apex + arc).
fn round_join_sector(apex: Point, n_in: (f64, f64), n_out: (f64, f64), delta_mm: f64) -> Polygon {
    let cross = n_in.0 * n_out.1 - n_in.1 * n_out.0;
    let dot = n_in.0 * n_out.0 + n_in.1 * n_out.1;
    let sweep = cross.atan2(dot); // signed turn angle, (-π, π]
    let mut pts = Vec::with_capacity(8);
    pts.push(apex);
    let mut vx = n_in.0 * delta_mm;
    let mut vy = n_in.1 * delta_mm;
    pts.push(Point::new(apex.x + vx, apex.y + vy));
    let steps = (steps_per_rad(delta_mm) * sweep.abs()).ceil() as i32;
    if steps > 1 {
        let step = sweep / f64::from(steps);
        let (s, c) = step.sin_cos();
        for _ in 1..steps {
            let nx = vx * c - vy * s;
            let ny = vx * s + vy * c;
            vx = nx;
            vy = ny;
            pts.push(Point::new(apex.x + vx, apex.y + vy));
        }
    }
    pts.push(Point::new(apex.x + n_out.0 * delta_mm, apex.y + n_out.1 * delta_mm));
    Polygon::new(pts)
}
```

```rust
fn minkowski_disc_primitives(ring: &[Point], delta_mm: f64) -> Vec<Polygon> {
    let n = ring.len();
    let mut prims = Vec::with_capacity(n * 2);
    let unit_normal = |from: Point, to: Point| -> Option<(f64, f64)> {
        let dx = to.x - from.x;
        let dy = to.y - from.y;
        let len = dx.hypot(dy);
        (len > f64::EPSILON).then(|| (dy / len, -dx / len)) // right-of-travel normal
    };
    for i in 0..n {
        let prev = ring[(i + n - 1) % n];
        let a = ring[i];
        let b = ring[(i + 1) % n];
        // Symmetric edge rectangle for edge a->b (unchanged behaviour).
        if let Some((nx, ny)) = unit_normal(a, b) {
            let off = delta_mm;
            prims.push(Polygon::new(vec![
                Point::new(a.x + nx * off, a.y + ny * off),
                Point::new(b.x + nx * off, b.y + ny * off),
                Point::new(b.x - nx * off, b.y - ny * off),
                Point::new(a.x - nx * off, a.y - ny * off),
            ]));
        }
        // Edge-aligned round-join sector at vertex a (incoming prev->a, outgoing a->b).
        if let (Some(n_in), Some(n_out)) = (unit_normal(prev, a), unit_normal(a, b)) {
            prims.push(round_join_sector(a, n_in, n_out, delta_mm));
        }
    }
    prims
}
```
